📊 Data Professional Survey Dashboard – Power BI

Author: Kareeb Sadab

📌 Project Overview

This project is a comprehensive Power BI dashboard created by Kareeb Sadab to analyze and visualize insights from a global Data Professional Survey. The dashboard provides a clear understanding of the professional landscape, salary trends, programming language preferences, and overall workforce satisfaction.

It demonstrates advanced data modeling, visualization, and analytical skills in Power BI, transforming raw survey data into actionable insights.

🎯 Key Insights & Visualizations
🌍 Countries of Survey Respondents

Geographic distribution of participants highlights global representation in the data industry.

💰 Average Salary by Job Title

Comparative salaries across roles like Data Scientist, Data Engineer, Data Analyst, Data Architect, and others.

Helps identify compensation trends and career growth opportunities.

👥 Survey Participant Summary

Total Respondents: 630

Average Age of Voters: 29.87 years

⚖ Work-Life Balance Satisfaction

Gauge visualization showing average happiness with work/life balance (5.74/10).

💵 Salary Happiness

Gauge visualization reflecting average satisfaction with salary (4.27/10).

💻 Favorite Programming Languages

Most preferred languages among survey respondents, highlighting industry tools demand (Python, R, SQL, etc.).

🚪 Difficulty to Break Into Data

Donut chart showing perceived barriers for entering the data field, from "Very Easy" to "Very Difficult".
